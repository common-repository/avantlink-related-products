Outputs a widget of related products from an AvantLink Product Search query based on a set of site-wide or post-specific keywords.

== Description ==

AvantLink Related Products is a tool for connecting Wordpress and an Avantlink Affiliate account. Given a set of site-wide or post-specific keywords, this plugin executes an AvantLink Product Search query and creates a widget of related products for display. In addition to providing site-wide or post-specific support of keywords, AvantLink Related Products also supports negative keywords and control over the number of results to display.

Developed by [CleanSnipe.com](http://cleansnipe.com).

== Installation ==

1. Upload the `avantlink-related-products` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place `<?php if (function_exists('display_related_products')) { display_related_products(); } ?>` within The Loop of your `single.php` or `page.php` files where you want the Related Products widget to appear.
4. Navigate to the 'AvantLink Related Products' page within the 'Plugins' menu in WordPress and adjust the plugin's settings as desired.
5. Define keywords within the "AvantLink Related Products" meta box of specific posts or pages.
